## How to install 

-   Clone Repo
```sh
$ git clone https://github.com/jack5341/react-search-engine.git
```

- Install Dependencies
```sh
$ npm install 
```

or

```sh
$ yarn
```
> Same steps for Server Side "./server"
- Build
```sh
$ npm run build
```

or 

```sh
$ yarn build
```
